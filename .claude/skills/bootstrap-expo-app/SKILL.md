# Bootstrap Expo App

Build the smallest mobile skeleton that can prove the product loop:

1. app shell and navigation,
2. auth entry and signed-in state,
3. one core home screen,
4. one settings/profile surface, and
5. enough shared design tokens to avoid repainting everything later.

Do not treat the first pass as a feature-complete mobile roadmap.
