# Launch Landing Page

Build the smallest credible landing page that can:

1. explain the problem and audience clearly,
2. collect interest with a concrete CTA,
3. capture qualitative feedback, and
4. give the team signal on what message resonates.

Prefer a homepage, waitlist form, and one lightweight feedback path over a larger marketing site.
